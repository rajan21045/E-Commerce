# Ecomm

A simple E-Commerce Shopping Store project built using PHP and MySQL for learning and practice purposes.

## About The Project

This project was created during PHP classes to learn how real-world web applications are developed using PHP. The project demonstrates the basic functionalities of an online shopping system, including user management, product handling, shopping cart operations, and admin features.

This repository is uploaded for educational and practice purposes.

## Features

* User Registration & Login
* Admin Panel
* Product Management
* Shopping Cart System
* Order Management
* Database Integration
* Responsive E-Commerce Interface

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* XAMPP

## Installation Guide

Follow these steps to run the project locally:

1. Install XAMPP
2. Start Apache and MySQL from XAMPP Control Panel
3. Copy the project folder into the `htdocs` directory
4. Open phpMyAdmin
5. Create a new database
6. Import the `ecommerce.sql` file
7. Open your browser and run the project using:

```bash
http://localhost/Ecomm
```

Database file included:
